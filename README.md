# AVL Tree (C++)

## Описание

Данный проект представляет собой реализацию самобалансирующегося бинарного дерева поиска (AVL-дерева) на C++ с использованием `std::unique_ptr`.

В реализации используются следующие принципы и решения:

* управление памятью через RAII (ресурсы освобождаются автоматически)
* отсутствие утечек памяти
* разделение на структуру узла (`Node`) и структуру дерева (`AVLTree`)
* автоматическая балансировка после вставки и удаления
* поддержка чисел с плавающей точкой через `epsilon`
* наличие обходов и визуализации дерева

Сложность основных операций:

| Операция | Сложность |
| -------- | --------- |
| Вставка  | O(log n)  |
| Удаление | O(log n)  |
| Поиск    | O(log n)  |

# Использование

### Создание дерева

```cpp
AVLTree<int> tree;
```

---

### Вставка

```cpp
tree.InsertNode(10);
```

Если элемент уже существует, он не будет добавлен.

---

### Удаление

```cpp
tree.RemoveNode(10);
```

---

### Поиск

```cpp
bool exists = tree.ContainsNode(5);
```

---

### Обходы

```cpp
tree.printInOrder();
tree.printPreOrder();
tree.printPostOrder();
```

---

### Визуализация

```cpp
tree.printVertical();
tree.printHorizontal();
```

---

## Свойство бинарного дерева поиска

Для каждого узла выполняется:

```
        node
       /    \
   left     right
```

Все значения в левом поддереве меньше значения узла,
все значения в правом поддереве больше значения узла.

---

## Балансировка AVL

Для каждого узла вычисляется фактор баланса:

```
balance = height(right) - height(left)
```

Допустимые значения:

```
-1, 0, 1
```

Если значение выходит за пределы, выполняется поворот.

## Вставка

1. Элемент вставляется как в обычное бинарное дерево поиска
2. При возврате из рекурсии обновляется высота
3. Выполняется балансировка через повороты

---

## Удаление

Рассматриваются три случая:

### 1. Узел без детей

Узел просто удаляется

---

### 2. Один ребёнок

Узел заменяется своим ребёнком

Пример:

```
 10
 /
5
```

После удаления 10:

```
5
```

---

### 3. Два ребёнка

1. Находится минимальный элемент в правом поддереве
2. Значение копируется в текущий узел
3. Минимальный элемент удаляется

Пример:

До:

```
     10
    /  \
   5    15
       /
      12
```

После:

```
     12
    /  \
   5    15
```

---

## Работа с числами с плавающей точкой

Для корректного сравнения используется значение:

```cpp
static constexpr double epsilon = 1e-9;
```

Сравнения выполняются через функции:

* `equal(a, b)`
* `less(a, b)`

Это позволяет корректно работать с числами типа `float` и `double`.
